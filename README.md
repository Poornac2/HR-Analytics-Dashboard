# 📌 Project Background

Human Resource Analytics helps organizations understand workforce trends, employee performance, and attrition patterns through data-driven decision-making.

This HR Analytics Dashboard was developed using Microsoft Excel to transform raw employee data into meaningful business insights. The dashboard enables HR professionals to monitor workforce metrics, identify high attrition areas, and support strategic HR planning.

---

# 🎯 Project Objectives

- Analyze employee attrition trends.
- Understand workforce demographics.
- Compare department-wise employee performance.
- Evaluate job satisfaction and employee distribution.
- Build an interactive HR dashboard using Microsoft Excel.
